Tic Tac Toe

Object Oriented Programming | Python | Minimax Algorithm
July 2021

Project Description

Tic Tac Toe is a classic board game. I decided to build my own version of the game using pygame GUI. The game can be played in three modes: Human vs. Human, Easy AI vs. Human, and Hard AI vs. Human. For Hard AI vs. Human, I implemented minimax algorighm to maxmizie the chance of the computer player to win.

Skills

Object oriented programming, minimax algorithm, recursive

Tools

Python, NumPy, pygame

Minimax Algorithm

Minimax is a backtracking algorithm used in decision making and game theory to find the optimal move for a player, assuming opponent player plays optimally. It is widely used in two player turn-based games. The two players are called maximizer and minimizer. The goal of players is to minimize the opponentâ€™s benefit and maximize self-benefit. The Minimax algorithm conducts a depth-first search to explore the complete game tree and then proceeds down to the leaf node of the tree, then backtracks the tree using recursive calls.
